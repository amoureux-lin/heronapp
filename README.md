# 🎮 heron App

这是一个基于 Flutter 构建的App 支持多语言、主题切换、动态内容、页面状态管理与拖动按钮等功能。采用 `Riverpod` + `GoRouter` 构建模块化结构，适用于 Web 和移动端。
